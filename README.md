My Smart Daily Planner

Description:
A web-based daily planner that helps users organize tasks easily. Users can add tasks, set priorities, mark them done or undo, delete tasks, filter by priority or status, and drag & drop tasks to reorder. Tasks are saved in the browser so they stay after refreshing.

Features:
 • Add tasks with title and priority (High, Medium, Low)
 • Mark tasks Done or Undo
 • Delete tasks
 • Filter tasks: All / High / Medium / Low / Done / Pending
 • Drag & drop to reorder tasks
 • Tasks are saved in the browser (localStorage)
 • Motivational quote displayed at the top

How to Use:
 1. Open index.html in a browser.
 2. Add a task and select priority, then click Add Task.
 3. Use Done/Undo to mark completed tasks or undo.
 4. Use Delete to remove tasks.
 5. Click filter buttons to view tasks by priority or status.
 6. Drag tasks to reorder them.

Technologies Used:
 • HTML
 • CSS
 • JavaScript
